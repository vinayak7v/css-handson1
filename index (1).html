What is CSS ?

  1. CSS is Stand for Cascading Style Sheets

  2. CSS Discribe how HTML element are to be displayed on screen,paper,or in other media

  3. Css is not Neccesary Css saves a lots of Work .It can control the layout of multiple Web pages all at onces

  4. External stylesheet are stored in CSS files

CSS can be added in HTML document in 3 ways:

  • Inline - by using the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external css file.
  • The most common way to add css, is to keep the styles in external CSS file. However, in this
    tutorial we will use inline and internal style,     because this is easier to demostrate, and easier for you to it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a styles to multiple elements.ID,on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
